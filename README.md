# My CLI Tool

这是一个命令行工具项目，旨在提供简单易用的命令行功能。

## 安装

要安装该工具，请使用以下命令：

```
npm install -g my-cli-tool
```

## 使用

安装完成后，您可以在命令行中使用以下命令：

```
my-cli-tool <command> [options]
```

### 可用命令

- `command1`：描述命令1的功能。
- `command2`：描述命令2的功能。
